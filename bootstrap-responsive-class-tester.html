Responsive Class Tester

Bootstrap viewport sizes break down into three categories: ‘Phone’ (less than 768px), ‘Tablet’ (979px to 768px), and ‘Desktop’ (greater than 980px). By using the utility classes .visible-phone, .visible-tablet, and .visible-desktop, you can show elements based on the viewport size. Conversely, you can hide elements with .hidden-phone, .hidden-tablet, and .hidden-desktop. Resize your browser to watch these test cases change.
